# Chuks Kitchen

## Project Overview

Chuks Kitchen is a food ordering website where users can explore meals, sign in, sign up, and place food orders through a clean and simple user interface. The project is built using only **HTML, CSS, and JavaScript**.

## How to Run the Project

1. Clone the repository:

```

```

```

3. Open `index.html` in your browser to start using the website.
